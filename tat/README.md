# @tryangletree/tat

TryAngleTree language package.

## Features

- Tokenize TAT source
- Parse TAT source into an AST
- Print AST output for inspection
- Export low-level lexer/parser utilities
- Foundation for runtime graph execution

## Public API

- `parseTat`
- `tokenizeTat`
- `parseTatToAst`
- `printTatAst`
- `tokenize`
- `parse`
- `printAST`
- `ParseError`
